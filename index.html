Rule 1
Kin Selection
r > c/b — Hamilton's Rule (1964), SOM eq. 2
Population is well-mixed (every generation each agent is paired with 8 random partners — kin selection does not require spatial structure, only relatedness). Each pairwise game uses the exact inclusive-fitness payoff matrix derived in the SOM: C-C=(b-c)(1+r), C-D=br-c, D-C=b-rc, D-D=0, i.e. my fitness = my own payoff + r × my partner's payoff. This matrix makes cooperators strictly dominant exactly when b/c > 1/r (equivalently r > c/b). Strategy updates follow a Fermi imitation rule: an agent copies the fitter of 8 random individuals with probability rising with the fitness gap and selection strength ω.

Rule 2
Direct Reciprocity
w > c/b — w = prob. of next round, SOM eq. 4
Well-mixed population playing the repeated Prisoner's Dilemma with continuation probability w. Strategies: AllC, AllD, TFT, GTFT (forgives with probability 1−c/b, exactly as in the main text). Payoffs are the exact closed-form sums of the underlying geometric series — e.g. TFT vs TFT = (b-c)/(1-w), TFT vs AllD = -c, AllD vs TFT = b — reproducing the SOM's 2×2 matrix exactly rather than approximating it by simulating a capped number of rounds. TFT/GTFT can resist invasion by AllD exactly when w > c/b.

Rule 3
Indirect Reciprocity
q > c/b — q = prob. of knowing reputation, SOM eq. 5
Well-mixed population of two strategies: discriminators (help unless they know the recipient is a defector) and defectors (never help). With probability q the donor correctly learns whether the recipient truly is a defector and refuses to help them — this exactly reproduces the SOM matrix C-C=b-c, C-D=-c(1-q), D-C=b(1-q), D-D=0 in expectation, at every generation, not just at equilibrium. The green "Avg Reputation" readout is a separate, cosmetic running average of each agent's recent realized cooperation (the "graded image score" the SOM mentions as the harder, unsolved case) — it tracks behavior for display but does not feed back into the game itself, so it can never desynchronize the dynamics from the exact matrix above. Cooperation is stable exactly when q > c/b.

Rule 4
Network Reciprocity
b/c > k — k = avg. number of neighbors, ref. 41 (Ohtsuki et al. 2006)
Agents occupy a lattice and interact only with their k neighbors. A cooperator pays cost c for each neighbor to receive benefit b; defectors pay nothing and give nothing — the literal cost/benefit rule from the main text. Updating follows the SOM's evolutionary-graph-theory process exactly: a random individual dies, and its neighbors compete to fill the vacancy proportional to fitness 1−ω+ωP. Pair-approximation under this process shows cooperators reach fixation more often than 1/N exactly when b/c > k, for weak selection (small ω) — try raising ω to see the prediction degrade outside that regime.

Rule 5
Group Selection
b/c > 1 + n/m — n = max group size, m = # of groups, SOM section 5
Literal implementation of the SOM's two-level model: the population is split into m groups, each capped at size n. Each tick, one individual from the entire population is chosen to reproduce proportional to fitness 1−ω+ωP (P from the in-group game C-C=b-c, C-D=-c, D-C=b, D-D=0); its offspring is added to its own group. If that group has just reached size n, with probability p it splits in two and a randomly chosen group elsewhere goes extinct; otherwise a random member of that (now full) group dies. Lower-level selection always favors defectors, but higher-level selection between groups favors cooperators — net cooperation wins exactly when b/c > 1+n/m.

⬡ UNIFIED SIMULATION — All Five Rules, One Coherent Engine

This is not in the paper — Nowak analyzes each mechanism separately. But all five reduce, in the SOM, to a 2×2 payoff-matrix transformation of the same baseline game C-C=b-c, C-D=-c, D-C=b, D-D=0. Here every mechanism is wired in as exactly that same transformation, applied simultaneously and compositionally, so that switching any four off reduces the model exactly to the one remaining rule's standalone matrix.

payoff(i,j) = repeated-game value of (effStrategy_i, effStrategy_j) with continuation w (Direct, off ⇒ w=0)
effStrategy_i = ALLD if Indirect-on and (prob q) i correctly learns j truly is a defector, else strategy_i (Indirect)
fitness_i += payoff(i,j) + r·payoff(j,i) for each partner j   (Kin, off ⇒ r=0)
partners j = lattice neighbors (Network on) | random same-group mates (Group on, Network off) | random global (both off)
reproduction = Fermi imitation of the fittest partner; groups additionally compete border-vs-border (Group on)
